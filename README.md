# JavaScript-Basics
<h1 align="center">JavaScript-Basics</a><h1>
Реализуйте функцию fib() и экспортируйте её по умолчанию. Функция вычисляет положительные числа Фибоначчи. Аргументом является порядковый номер числа.

Формула:
```js
    f(0) = 0
  f(1) = 1
  f(n) = f(n-1) + f(n-2)
  fib(3);  // 2
  fib(5);  // 5
  fib(10); // 55
```
